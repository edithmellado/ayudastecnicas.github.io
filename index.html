ADAPTADOR / SOPORTE PARA CORTAUÑAS

El soporte para cortauñas es una herramienta que hace que cortar las uñas sea una tarea menos extenuante para personas con poca destreza. Tiene dos partes separadas, un cuerpo para sujetar el cortauñas y un mango para presionar el cuerpo hacia abajo con la mano, el soporte cuenta con áreas de rebaje para indicar dónde debe ir pegada la goma eva, la cual funciona como antideslizante.

CONCLUSION

A partir del primer modelo impreso, se fue analizando y definiendo de mejor manera tanto al usuario como a la problemática a suplir, hasta llegar a algo que ayude a las personas a cortar fácilmente las uñas de los pies sin necesidad de levantar las piernas o doblar la cintura, permitiéndole al usuario tener mejor autonomía y ser más independiente.
